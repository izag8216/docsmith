# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-04-29

### Added

- Initial release
- AST-based Python source parser
- Support for Google, NumPy, and Sphinx docstring formats
- Jinja2 template engine for customizable output
- CLI with `generate`, `single`, `serve`, `diff`, and `init` commands
- Watch mode for live reload
- Documentation coverage diff
- 80%+ test coverage
- i18n support (English and Japanese)
